Analysis of 'seva.org'  website

Analysis was conducted based on CRAP principles:

Contrast

Size Contrast: font sizes to indicate information hierarchy

 

Color Contrast: Dark text on light backgrounds (or vice versa) - Accent colors for links, buttons, and key information - Sufficient contrast ratios for accessibility compliance

Font Contrast:  Bold headings vs. regular body text - Different font families for headings vs. body

Spatial Contrast:  Dense information areas vs. white space - Varied spacing between elements -

Different content block sizes

Repetition 

Typography Patterns:  Consistent heading hierarchy (H1, H2, H3 styling) - Regular font choices
for body text - Standardized treatment of links and emphasis

 

Color Schemes: Consistent color palette throughout the site 

Repeated accent colors for similar
functions - Standardized background and text color combinations

Spacing and Layout:  Consistent margins and padding - Standardized spacing between elements
- Repeated layout patterns for similar content types

Visual Elements: Consistent bullet point styles - Standardized image treatments and captions -
Repeated icon or graphic styles

Alignment 

Left Edge Alignment: aligned text for easy reading (most common for body content)

Right Edge
aligned elements for visual balance

OR

- Center-aligned text for titles and special emphasis

Visual Alignment:  Aligning images with text baselines - Consistent positioning of headings
relative to content - Aligned columns in multi-column layouts

Conceptual Alignment:  Related information grouped and aligned together -

Hierarchical
information aligned to show relationships - Sequential information aligned to show progression

Proximity

Visual Grouping:  Related text elements positioned close together - Headings placed near their
corresponding content - Supporting information clustered with main points

 

Spatial Separation:  White space used to separate different topics - Clear breaks between content
sections - Distinct spacing for different types of information

Functional Grouping:  Navigation elements grouped together - Related links clustered in specific
areas - Contact information or metadata grouped consistently
